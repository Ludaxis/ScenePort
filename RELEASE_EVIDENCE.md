# ScenePort Release Evidence

This file is **generated at release time** by `scripts/collect-release-evidence.mjs` (run in
the Release workflow) and uploaded as the `release-evidence` CI artifact and appended to the
GitHub Release notes. It is intentionally not hand-maintained here — any version number,
gate result, or commit captured in the repo copy would go stale.

To view the evidence for a given release, see the `release-evidence` artifact on that
release's CI run, or the evidence block appended to the GitHub Release body.

To regenerate locally:

```bash
node scripts/collect-release-evidence.mjs
```
